void ritcs::Bike::brake ( unsigned int  sec)

Hit the brakes on the bike for a certain number of seconds.

The speed of the bike is decreased by the current gear times the number of seconds. In the event the speed of the bike drops to or below zero, the speed should be set to 0 and the bike should be put into first gear.
